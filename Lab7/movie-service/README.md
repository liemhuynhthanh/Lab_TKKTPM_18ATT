# Movie Service (Node.js + Express)

Part of the Movie Ticket System (Event-Driven Architecture).
## API Endpoints
- `GET /movies`: List all movies.
- `POST /movies`: Add a new movie.
- `PUT /movies/:id`: Edit a movie.


The service will run on port `8082`.
